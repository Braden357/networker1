# Networker

Student networking product (spec and plans at repo root). **Phase 1 web app** lives in [`web/`](web/README.md).

- [Development plan](DEVELOPMENT_PLAN.md)
- [Project spec](PROJECT_SPEC_FROM_INTERVIEW.md)

## Quick start (Next.js)

```bash
cd web
npm install
cp .env.example .env.local
# Add Supabase URL + anon key, then:
npm run dev
```

See [`web/SUPABASE_SETUP.md`](web/SUPABASE_SETUP.md) and [`web/README.md`](web/README.md) for Supabase, migrations, and Vercel.
